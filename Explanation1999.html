Principles Behind 1999's User Interface


Three-column layouts are here! Websites incorporated more user interactivity as Web 2.0 gained more traction HTML 4.0 had been available for a year. Search mechanisms and site maps started emerging; not necessarily because websites were complicated enough to need them, but because they were easy to implement. Layouts were table-based. I didn't use a table-based layout when creating the 1999 page, however. The columns of CSS grid made mimicing a three-column layout much easier!
