2014's design is grid-based. Specifically, each row on the page spans a certain number of columns with each column's length taking up 1/12 of the screen's width. This allows for easier implementation of responsive design. Websites are now starting to adopt a "mobile first" methodology where they are designed for the smallest screen widths first. Navigation menus replace navigation bars for a more optimal mobile experience and designs generally use all of a screen's width.
