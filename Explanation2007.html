Principles Behind 2007's User Interface

Though the smartphone became a popular medium through which a user could access the internet, websites had not designed their pages for mobile browsers yet. After 2004, table-based layouts were replaced with <div> based layouts. <div> layouts were faster, more flexible, and easier to maintain. Despite this change, the user interfaces did not seem to change much. As such, I elected to keep the table-based layout from 2004 and make minor changes to it since I do not need to "maintain" the pages on this website. Search bars became more prominent as users relied on search functions more, thus causing site maps (not to be confused with sitemaps) to become less common.
