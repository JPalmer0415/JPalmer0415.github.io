This webpage attempts to mimic the user interface of a corporate website's homepage in 1997. 1997 was the first year with HTML, Javascript, and CSS. The importance of providing a good UI increased as the number of internet users increased, thus web design was starting to become a serious consideration. However, there were still no standards for UI design. The consistencies across corporate websites that I could find were the use of a navigation pane on the left, icons, footers to help navigate websites, and tables to structure and separate content.
