Principles Behind 2004's User Interface

Web 2.0 became popular in 2004, shifting web design towards encouraging user participation. Layouts were still table-based, but now used XHTML. Search bars became more prominent as users generally now knew how to look things up. Navigation bars, rounded edges, and shadows gain prominence as well. The general aesthetic of 2004 was skeuomorphism, an aesthetic that attempts to invoke a sense of familiarity in viewers by mimicing real-world objects in digital interfaces.
